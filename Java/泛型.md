## 泛型
泛型程序设计意味着编写的代码可以对多种不同类型的对象重用


### 一. 泛型方法 
- 类型变量放在修饰符后面(例如public private)，在返回类型的前面
- 可以在普通类型中定义泛型方法，也可以在泛型类型中定义
- 当调用一个泛型方法是，把具体烈性包围在尖括号中，放在方法名前面

针对泛型方法做一个示例和声明的说明:
```java

/**写了一个ArrayAlg的声明*/

class ArrayAlg{

    // 其中尖括号的<T> ,位置是出于修饰符 和 返回类型之间 ，是表示泛型参数的声明，告诉编译器，这个方法使用一个泛型类型T
    // 第二个T代表函数的返回类型
    public static <T> T getMiddleEle(T... a){
        return a[a.length / 2];
    }

}
```
